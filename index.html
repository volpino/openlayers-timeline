OpenLayers Timeline Demo

OpenLayers Timeline is a Javascript library created by Federico "fox" Scrinzi (fox91 at anche dot no).

It gives to OpenLayers the power to display data in both spatial and temporal context.

It currently supports these formats for data source (click on the links to see a demo):

  • GeoJSON demo (data from Toolserver)
  • GeoRSS demo (data from Flickr)
